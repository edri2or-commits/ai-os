## Usage Instructions

When generating a Drive snapshot:

1. **Copy this template** to Google Drive folder "AI-OS System State"
2. **Rename** to `SYSTEM_SNAPSHOT_DRIVE.md`
3. **Fill in values** from:
   - `docs/SYSTEM_SNAPSHOT.md` (primary source)
   - `docs/CONTROL_PLANE_SPEC.md` (mode, phase)
   - `docs/system_state/AUTOMATIONS_REGISTRY.jsonl` (automations)
   - `agents/AGENTS_INVENTORY.md` (agent details)
   - `docs/PHASE2_CHECKLIST.md` (phase progress)
   - `logs/EVENT_TIMELINE.jsonl` (recent events, if exists)
4. **Update Metadata** with generation time, generator, confidence
5. **Save** â€” this becomes the "view" for GPT Planning Model

**Update Triggers:**
- End of significant work session
- Phase transition
- After major repo changes
- Weekly refresh (if no other updates)
- On request from Or or any agent
