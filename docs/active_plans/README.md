# 📂 Active Plans Directory

תיקייה זו מחזיקה תוכניות פעילות שטרם נסגרו.

כל קובץ כאן מייצג תוכנית מערכתית פעילה (למשל: Phase 2 – Stabilizing the Hands).

כאשר התוכנית נסגרת (`status: closed`), הסקריפט `scripts/move_completed_plan.py` מעביר אותה אוטומטית לתיקיית `docs/completed_plans/`.

סוכנים חדשים עוברים דרך תיקייה זו בזמן החיול, כדי לזהות אם קיימת תוכנית פעילה שעליהם להצטרף אליה.
