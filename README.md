# AI-OS – מערכת ההפעלה האישית שלי ל-AI

הריפו הזה הוא "המוח המרכזי" של מערכת ה-AI האישית שלי.

## המטרות:
- ליצור מקור אמת אחד (Single Source of Truth) לכל החוקים, הסוכנים והכלים.
- לאפשר סנכרון בין מודלים וצ'טים שונים (Claude, GPT וכו') דרך מסמכים משותפים.
- לבנות מערכת תשתית יציבה, שניתן להרחיב אותה בצורה מבוקרת (Thin Slices).

## מבנה הריפו:
- `docs/` – מסמכי הסבר, חוקה, ומיפוי של המערכת.
- `agents/` – הגדרות סוכנים (תפקיד, גבולות, יכולות).
- `tools/` – חיבורים לכלים חיצוניים (Google, Telegram, Make, GitHub, GPT API וכו').
- `workflows/` – תהליכי עבודה: מי מפעיל מה, ומה סדר הצעדים.
- `policies/` – חוקים, הרשאות וגבולות פעולה.
- `archive/` – קבצים שנמשכים מהריפו הישן כדי לבדוק אם הם עדיין רלוונטיים.

## בהמשך:
- נוסיף תיעוד לכל סוכן וכל כלי.
- נגדיר תהליכים קטנים שעובדים מקצה לקצה שישמשו אותי ביום־יום.
