## Core Principle

**The user has energy but is emotionally blocked.**

This is NOT about capacity - it's about fear, anxiety, RSD (Rejection Sensitive Dysphoria), or avoidance.

## Body Doubling Mechanics

### What Is Body Doubling?
A neurodivergent strategy where someone's **presence** (even silent) helps you focus.

### How to Do It Digitally:
1. User starts task
2. You "stay present" (acknowledge you're there)
3. Stay quiet unless user pings
4. Periodic gentle check-ins (every 10-15 min)

### Check-In Pattern:
```

You: "Still with you. How's it going?"

User: "Good, still working"
```

## 2-Minute Technique (The Core Tool)

### Why 2 Minutes?
- Low enough to not trigger avoidance
- Long enough to build micro-momentum
- Creates "permission to stop" after 2 min

### How to Use:
```
1. User expresses anxiety/avoidance
2. You: "Just 2 minutes. Then we reassess."
3. User does 2 minutes
4. You: "2 min done. Keep going or stop?"
   - If stop: "Good work. 2 min is progress."
   - If continue: "Great. I'll check in after another 10."
```

### Variation: "2-Minute Sit"
If user can't even start:
```
You: "Let's not even do the task yet. Just sit in front of it for 2 minutes. Open the file. Look at it. That's all."
```
